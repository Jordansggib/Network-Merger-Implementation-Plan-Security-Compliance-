# Network-Merger-Implementation-Plan-Security-Compliance-
Secure hybrid cloud network merger plan for a financial firm acquiring a healthcare software provider. Implements Zero Trust and defense in depth. Ensures PCI DSS and HIPAA compliance within a 50,000 dollar budget.


NETWORK MERGER AND IMPLEMENTATION PLAN

OBJECTIVE
Merge Company A and Company B networks. Plan integrates on premises and cloud systems.

ISSUES
Companies run end of life systems. Security gaps exist. Lack of access control creates risk. Legacy hardware lacks support.

SOLUTIONS
Replace Cisco and FortiGate units. Deploy MikroTik and Aruba hardware. Move server workloads to Azure. Upgrade workstations to Windows 11.

SECURITY
Use Zero Trust. Users authenticate at every connection. Apply defense in depth. Enforce Multi Factor Authentication.

COMPLIANCE
Meet PCI DSS for financial data. Support HIPAA for medical software. Azure Cloud ensures data protection.

BUDGET
Project cost stays under 50,000 dollars.

SKILLS
Network Design. Risk Assessment. Zero Trust. Cloud Migration. Compliance Management.
